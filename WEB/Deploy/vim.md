# 三种工作模式

## 命令模式

打开文件首先进入命令模式，是使用 `vi` 的入口 
通过命令对文件进行常规的编辑操作，例如：定位、翻页、复制、粘贴、删除…… 
在其他图形编辑器下，通过快捷键或者鼠标 实现的操作，都在命令模式下实现

## 末行模式 
执行 保存、退出 等操作 

要退出 `vi` 返回到控制台，需要在末行模式下输入命令 
末行模式是 `vi` 的出口 
```vim
w  保存 
q  退出，如果没有保存，不允许退出 
q! 强行退出，不保存退出 
wq 保存并退出 
x  保存并退出
```
